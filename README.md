# Char_Maker

Desenvolvendo um projeto de editor de personagem no estilo de jogos Dress Up e do Site Picrew, com enfoque em criar persoangens de RPG

Diferenciais do projeto:
    Escolha de cores dinamica, com possibilidades infinitas de coloração;
    Dinamismo nas regras de cor, permitindo que partes anteriormente ligadas nas cores se desvinculem;
    Regra de bloqueio para partes que não deveriam estar juntas;
    Perfis de personagem (Normal, pequeno e grande);
    Save de projeto e abertura, para facilitar mestres gerenciar multiplos personagens;

Entre outras funções ainda a serem implementadas
